# 🏨 Hotel Booking Cancellation Analytics & Revenue Optimization Engine

An elite-tier data science and business intelligence portfolio project designed to mitigate high cancellation rates, minimize revenue leakage, and maximize room utilization efficiency for City and Resort hotel properties. 

Rather than relying on generic, automated machine learning pipelines, this project implements a highly tactical, domain-driven analytical framework to deliver actionable financial strategies to hotel stakeholders.

## 🎯 Project Core Pillars
* **Domain-Driven Data Profiling:** Preserved the long-tail lead time data (>365 days) to explicitly capture macro-booking risks instead of wiping them out with blind statistical clipping.
* **Advanced Feature Engineering:** Formulated a custom `Revenue Risk` metric ($ADR \times Total\ Nights$) and a behavioral tracking `Flakiness Index` to flag repeat-offender customer segments.
* **Statistical Inference:** Conducted rigorous non-parametric hypothesis testing (Mann-Whitney U) and categorical independence tests (Chi-Square) to map pricing sensitivity and deposit vulnerabilities.
* **Financial Decision Simulation:** Programmed an interactive, data-driven ROI policy simulator to project the net revenue preserved by enforcing strict non-refundable deposit terms on high-risk booking windows.
